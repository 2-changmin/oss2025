### italics체로 글자 쓰기
*italics* ( \*내용\*) <br>
_italics_ ( \_내용\_) <br>

\*literal asterisks\*  (특수문자같은걸 문자 그대로 나타내기 위한 용도) <br>

### bold체로 글자 쓰기
**bold** (\*\*내용\*\*) <br>
__bold__ (\_\_내용\_\_) <br>

### 글자 중간에 선 긋기
~~strikethrough~~ (\~\~내용\~\~) <br>

### 순서 매기기

2. Second item
   1. Subitem

   + item 3
